点击按钮，复制指定文本框内容，代码如下：

<!-- more -->
``` html
<script type="text/javascript">
function copy(){
    var str = document.getElementById("my-data");
    str.select(); // 选择对象
    document.execCommand("Copy"); // 执行浏览器复制命令
    alert("复制成功！");
}
</script>

<textarea cols="20" rows="10" id="my-data">我的测试内容</textarea>
<input type="button" onClick="copy();" value="复制" />
```
